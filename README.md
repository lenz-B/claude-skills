# lenzb-claude-skills

A collection of Claude skills built for personal brand and content operations.

## Skills

- **anti-ai-writing** — strips AI patterns from any written output
- **deep-research** — multi-source research with playbook creation
- **design-taste-frontend** — senior UI/UX engineering standards
- **full-output-enforcement** — forces complete, untruncated outputs
- **skill-creator-enhanced** — build, test, and improve Claude skills

## How to use

Each skill lives in its own folder. Drop the folder into `/mnt/skills/user/` on your Claude setup and add the trigger description to your skills list.
